The Canonical Trick Play

You can download all of the SGFs used in this post here. I use Sabaki as my editor, which features some extra bling on the game tree. So, in order to extract the most from my SGFs, I advise you to use it too. Don't forget to donate to the project too!

The closest I got to finding a successful professional use of this trick play is a game between Kang Dongyun and Lee Jaeung, back in 2007, when they were both 5p. Kang Dongyun 9p uses this pattern to both enclose the corner and switch to developing the left side, something which wouldn't have been possible with the standard joseki in this case.
